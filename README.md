# Hydruscoin
一个简单支持UTXO模型和Account模型的数字货币,基于Hyperledger Chaincode。主要用于普及智能合约,非商用。

## 目录
* coin: 货币代码,诸如coinbase,转账等功能
* client: 简单的一些tx生成函数

## 实现功能
* coinbase交易
* 数字货币转账
* 账户查询
* 交易查询
* 数字货币简单统计
